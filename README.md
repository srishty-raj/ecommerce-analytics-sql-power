# E-commerce Analytics Dashboard (SQL + Power BI)

## Overview
This project analyzes e-commerce data to understand sales trends, customer behavior, and product performance. The goal is to derive actionable insights that can support business decision-making.

## Tools & Technologies
- SQL (data extraction & analysis)
- Power BI (dashboard & visualization)
- Excel (data cleaning)

## Key Business Questions
- What are the overall revenue trends?
- Which products generate the most revenue?
- How does customer behavior impact sales?
- What are the top-performing categories?

## Key Insights (Planned)
- Revenue growth trends over time
- High-performing products and categories
- Customer purchasing patterns

## Project Structure
- `/data` → dataset files
- `/sql` → SQL queries
- `/dashboard` → Power BI files/screenshots
